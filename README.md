# ngKeypad

A customizable keypad component made with Angular.js. Feel free to download and use.

## Demo
You can check it out [here](http://alvarowolfx.github.io/ng-keypad/).

## Installation

Install manually, or from bower:

```bash
$ bower install ngKeypad
```
